Programming during Studies

During my studies, I developed several applications, focusing on web development using modern technologies such as React for front-end and Django for back-end. These projects helped me deepen my understanding of full-stack development and improve my skills in creating dynamic, responsive, and scalable applications.

Coffeetime

CoffeeTime is my original web application designed to connect people for coffee meetings. It was initially created as part of a university project but later expanded from an idea for a café website in the dormitory. The application combines basic web technologies like HTML, CSS, and JavaScript, with PHP as the primary back-end language. Users can select convenient times for a coffee meeting, and the system automatically matches them and assigns a time within their availability. The idea came to me while... drinking coffee.

Simple Blog Management System

The Simple Blog Management System is a desktop application written in Python. It reflects my interest in creating tools that simplify tasks and automate work. The idea originated while working on a project involving writing posts for a website to optimize keywords for SEO. I needed a tool to easily manage the content I had already written, create thumbnails for posts, and check metadata for appropriate length. The application uses the Tkinter library with the ttkbootstrap wrapper for the GUI and implements a few design patterns I learned in one of my courses.

ReceiptRover

ReceiptRover is an OCR (Optical Character Recognition) application for scanning receipts and recording expenses, with the option for later analysis. It was developed by me and a group of colleagues as part of a Software Engineering course project. My primary contribution was creating the dynamic GUI, once again using the Tkinter library along with ttkbootstrap, which I had become familiar with from previous projects.
