Simple blog management system

Simple blog management system is a desktop application written in Python. It is designed based on my preferences, which include creating applications that simplify life and automate work. The idea for it arose in me during a project where I was tasked with writing posts for a website to optimize keywords for SEO. I needed a tool that would allow me to easily control the content I had already written, create thumbnails for posts, and check their metadata to ensure they are of appropriate length. The application uses the tkinter library with the ttkbootstrap wrapper for displaying the window. It utilizes a couple of design patterns that I learned in one of my courses.

Receiptrover

ReceiptRover is an OCR (Optical Character Recognition) application for scanning receipts and saving expenses with the ability to later analyze them. It was developed by me and a group of colleagues as a software engineering project. My main contribution to the application was creating the dynamic GUI (Graphical User Interface). Once again, I used the tkinter library along with ttkbootstrap, which I was already familiar with from a previous application.
